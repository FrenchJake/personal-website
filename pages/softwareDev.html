Software Development

Fast Enterprises, Denver CO

Title: Software Developer
Duration: August 2023 – June 2025

As part of a ground up product launch, I became a core developer on a brand new, enterprise level Child Support application. This system is designed to serve both child support professionals and the general public - including parents (obligors and obligees). I worked end-to-end on its development, helping translate real world government workflows into secure, scalable, and user friendly software.

  • Led development of full-stack features in C#, Java, and SQL within a collaborative Agile team focused on continuous delivery and rapid iteration.
  • Built and optimized data migration tools to convert legacy records into the new format, enabling seamless onboarding of entire state agencies.
  • Redesigned how Social Security Numbers were stored and processed using advanced encryption and tokenization, significantly elevating system wide privacy compliance.
  • Identified and resolved more than 200+ system level bugs during integration testing, improving system stability and becoming the team’s go to debugging resource.
  • Wrote highly efficient backend SQL queries to support large scale reporting and real time data visualization used by both internal staff and government officials.
  • Collaborated cross functionally with UI/UX designers, QA testers, and client stakeholders to build features that directly serve both agency workers and citizens.
  • Championed clean code standards and version control practices across the development team, leading peer reviews and contributing to long term maintainability.

What is Fast Enterprises?

Fast Enterprises develops and implements large-scale software solutions for government agencies. Their systems are used to manage functions like tax processing, motor vehicle registration, unemployment insurance, and child support enforcement. FAST specializes in replacing outdated legacy systems with modern, configurable software that improves efficiency, automates workflows, and ensures secure data management.
